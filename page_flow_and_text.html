Trust Game — Page Flow & Text

This document records, for the oTree experiment (trust_reinvestment app), every page each player sees in each cell, together with the verbatim English text of each page. Page text is copied straight from the templates ({{ }} placeholders preserved; the actual values used in this configuration are in the legend below).

Placeholder values (this configuration)
{{ C.ENDOWMENT }}Per-round endowment = 10 points
{{ C.MULTIPLIER }}Multiplier on the amount sent = 3
{{ C.STAGE1_ROUNDS }}Part 1 rounds = 3
{{ min_rounds }}Part 2 minimum rounds = 5
{{ stopping_probability }} / {{ continue_probability }}After round 5, each round stops = 15% / continues = 85% (hard cap 24 rounds, NUM_ROUNDS=27)
{{ payment_rate }}Per-point conversion = $0.10

Roles: player 1 / A = truster (sends first); player 2 / B = trustee (returns afterward). Pairs are fixed throughout.
